Readme CBD1400351 
 
 Workaround 
 
Using MICROSAR Safe Silence Verifier (MSSV) 
 
For errors in osekasm.h please refer to MicrosarOs_Rh850_SafetyManual, 
chapter 9.1.  
There you can find some instructions for apply MSSV.exe on the OS sources. 
For errors in Compiler_Cfg.h please start MSSV.exe with following parameters: 
    MSSV.exe -i <SourcePath> -i <GenPath> -i <HeaderPath> -D NOASM 
 
    Please use MSSV.exe in folder Generators/MSSV and ignore 
 
 
Generators/MSSV_legacy
